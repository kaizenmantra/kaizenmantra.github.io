Changing file explorer font size on Eclipse IDE

November 27, 2013 Shri

If you ever find yourself with a Mac on a rather large screen and your eyes are aching to get a better font size on the project explorer, try this...it works :-)

Navigate to the Eclipse->Plugins->org.eclipse.platform_x.y.z->css->Platform_Specific_CSS file

 

Copy the CSS file and keep a back up copy (just in case).

Now, add the following entry (I set my font size to 18), save the file, and restart Eclispe.

#org-eclipse-jdt-ui-PackageExplorer Tree, #org-eclipse-ui-navigator-ProjectExplorer Tree { font-size: 10px; /* <-- code="" desired="" font="" size="">

You should now have larger  font sizes in your explorer window like so.
