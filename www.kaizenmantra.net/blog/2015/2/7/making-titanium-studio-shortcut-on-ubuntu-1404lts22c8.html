Creating a launcher shortcut for Titanium Studio on Ubuntu 14.04 LTS

February 7, 2015 Saurabh

Ever wanted to install Ti Studio on Ubuntu. I recently switched one of my not so old laptop to Ubuntu because it was just too slow on Win7.

Being a newbie to Ubuntu, I spent far more time googling for stuff than I would have liked to. In any case, I had my Ti Studio installed and it worked fine...with just one caveat, there was no shortcut.

Yep, I had to navigate to /opt and launch it from there everytime. I tried doing a symlink but for some reason it did not work for me.

Finally, I stumbled upon this blogpost on Eclipse install and figured I can do the same for Titanium Studio. Well, it worked and here is how. Open terminal (Ctrl+Alt+T) and type in...

gksudo gedit /usr/share/applications/titanium.desktop

This will create a launcher file and open it with gedit. Paste the following code into this file..

[Desktop Entry]
Name=Titanium Studio
Type=Application
Exec=/opt/TitaniumStudio/TitaniumStudio
Terminal=false
Icon=/opt/TitaniumStudio/icon.xpm
Comment=Integrated Development Environment
NoDisplay=false
Categories=Development;IDE;
Name[en]=TitaniumStudio

Now, Titanium Studio can be launched from the Unity dash search...
